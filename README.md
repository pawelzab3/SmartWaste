
### CI/CD
Projekt obecnie nie wykorzystuje zaawansowanego CI/CD, ale można łatwo dodać GitHub Actions do:

- instalacji zależności  
- uruchamiania testów  
- budowania aplikacji  
- automatycznego wdrażania

# Podsumowanie

Dokumentacja opisuje:
- cykl tworzenia projektu,
- modele procesu,
- modelowanie UML,
- testowanie,
- architekturę,
- SCM i konfigurację,
- oraz wyjaśnia, dlaczego aplikacja jest sieciowa.

System SmartWaste to prosta webowa aplikacja, w której **wszyscy użytkownicy mają jednakowe prawa**, a dane przechowywane są w centralnej bazie MySQL.
